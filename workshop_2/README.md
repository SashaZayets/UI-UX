# “Моделювання логотипа в Figma”

## План виконання роботи

1. Створити фрейм
2. Створити 5 Rectangles та розмістити їх на фреймі
3. Налаштувати закруглення кутів у Rectangles
4. Покрасити Rectangles у відповідні кольори
