## Висновок

Для графічного зображення з текстом та скріншоту, найкраще підійшов WebP(lossy)(75%), дуже велике зменшення ваги файлу при хорошій якості.
Для фотографії без сильної втрати якості, найкраще підійшов AVIF (75%).
